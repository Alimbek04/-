Версия с ограничением до 25 студентов и фиксированным 1 днём посещаемости

Этот проект представляет собой консольное приложение для учёта посещаемости студентов.
Программа позволяет отмечать присутствие студентов за один день, выводить итоговый отчёт и сохранять данные в файл.


Функциональные возможности

 Ввод количества студентов (от 1 до 25)
 Фиксированное количество дней посещаемости: **1**
 Ввод имён студентов
 Отметка посещаемости (присутствует/отсутствует)
 Расчёт процента посещаемости за один день
 Сохранение данных в файл `attendance_log.txt`
 Отчёт по каждому студенту

Требования

 Установленный **.NET 6 SDK**
 Рекомендуемая среда: **Visual Studio Code**



Запуск

1. Откройте проект в Visual Studio Code.
2. Откройте терминал.
3. Выполните:

dotnet run


Как работает программа

1. Пользователь вводит количество студентов (до 25).
2. Количество дней автоматически установлено в 1
3. Пользователь вводит имена всех студентов.
4. Программа поочередно запрашивает посещаемость:

   
   Присутствует Иван? (y/n)
   
5. После ввода всех данных формируется отчёт.
6. Система сохраняет журнал посещаемости в файл `attendance_log.txt`.


Формат файла attendance_log.txt

Каждая строка содержит:


ИмяСтудента,True/False


Пример:


Иван,True
Мария,False
